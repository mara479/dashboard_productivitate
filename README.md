# dashboard_productivitate
Dashboard zilnic de productivitate în HTML/CSS/JS

## Link
https://mara479.github.io/productivity-dashboard/

## Descriere
Aplicația oferă o interfață modernă pentru organizarea activităților zilnice, fiind destinată îmbunătățirii productivității personale.

## Funcționalități
- adăugare task-uri cu titlu, categorie și deadline  
- filtrare task-uri pe categorii (Lucru / Învățat / Sănătate)  
- afișare progres zilnic (bară de progres și procent)  
- fereastră modală cu detalii task  
- temă light/dark cu salvare în LocalStorage  
- secțiune de review-uri salvate în LocalStorage  

## Tehnologii utilizate
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
